# Manga Shop

Финальный командный проект по frontend-разработке — интернет-магазин манги с полноценной клиентской частью, авторизацией, корзиной и оформлением заказов.

Frontend часть разработана на React + Vite с адаптивным дизайном и интеграцией с backend API.

# Features

* Регистрация и авторизация
* Каталог манги
* Поиск товаров
* Корзина
* Изменение количества товаров
* Оформление заказа
* Responsive / Mobile Design
* Работа с API
* LocalStorage для хранения токена
* React Router navigation

# Installation

Clone repository:

```bash
git clone https://github.com/gulfviruz/manga-shop-frontend-main.git
```

Open folder:

```bash
cd manga-shop-frontend-main
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
